SLIM

This is the workflow Slim supports:
  1. Write code.
  2. Run unit tests.
  3. Do a code review.
  4. Do an integration test. Passing this yields something for alpha testing.
  5. Do a system test. Passing this yields something for beta testing.
  6. Ship it! Create a version and deploy.
